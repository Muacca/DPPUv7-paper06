## 8. Discussion

### 8.1 What $\chi$ -universality organizes

本稿の結果は、Lorentzian EC+NY reduced sector において topology dependence が Hamiltonian admissibility・P-channel diagnostics・reduced vacuum orbit の三つのレベルにわたって同一の scalar $\chi$ によって組織されることを示す。

admissibility レベルでは、EH/AX/VT/MX の mode pattern が四つの topology に共通であり（Table 6）、topology の具体的な値は pattern 自体を変えない。P-channel レベルでは、 $P_{\rm form}$ が topology によらず exact zero となり（Section 4.2）、 $Q_{\rm NY}$ が mode separation を示すが（Section 4.4）、いずれも $\chi$ への依存を持たない。唯一 MX diagnostic $P_{\rm int}$ だけが $C_{\rm topology}=-9\chi$ を通じて topology を区別する（Section 4.3）。reduced-orbit レベルでは、auxiliary shell 上の方程式が $\dot q^2+\chi=0$ に帰着し（Section 7.1）、orbit の定性的分類が $\chi$ の符号で決まる（Section 7.2）。

これらを総合すると、四つの topology の違いは reduced sector において scalar $\chi$ の値の差異として現れ、それ以外の構造（admissibility pattern, $P_{\rm form}$ behaviour, mode separation）は topology によらず共通である。このことを $\chi$ -controlled topology universality と呼ぶ。

### 8.2 Relation to the paper04 response dictionary

paper04 [5] との比較が示唆するのは、DPPU を通じて用いられている四つの Thurston geometry が、単なる便宜的な example の列挙ではないという点である。paper04 では、これら四つの geometry が Euclidean KK Higgsing のもとで none / uniaxial / triaxial / biaxial の四 response type を尽くしていた。本稿の Lorentzian spin-0 / isotropic EC+NY reduction においては、同じ四つの geometry が five-parameter structure-constant family の構造的に区別される loci として再出現する。すなわち、原点（ $T^3$ ）、単一の class-A axis（ $Nil^3$ ）、等方的な class-A diagonal（ $S^3$ ）、および balanced な class-B pair（ $Sol^3$ ）である。したがって本稿の $\chi$ -universality 結果は、paper04 機構との直接的な同一視ではなく、同じ four-geometry response dictionary を別の reduced sector において読み直したものと位置付けられる。

これら二つの結果は、同一の Thurston geometry set が DPPU theory の異なる reduced sector で整合的な分類構造を与えることを示唆するが、その共通根拠の解明は将来の課題である。

### 8.3 Limitations and future directions

本稿は $\alpha=0$ EC+NY isotropic reduced sector に限定されており、以下の拡張は将来課題として明示的に残している。

$Nil^3$ および $Sol^3$ については、本稿では isotropic scale $q$ による reduced 記述のみを用い、独立した scale factors を持つ anisotropic EOM の導出は行っていない。 $Sol^3$ については compact quotient の構成、spin structure の分類、および spectral data は本稿の範囲外である。 $S^3$ の vacuum reduced atlas に実数初期データが存在しないことは、matter coupling または effective source を導入すれば解消され得るが、その解析は本稿の範囲外である。Weyl-source / $\alpha\neq0$ branch についても、本稿は $\alpha=0$ baseline に限定しており、それ以外の branch の再評価は含まない。

$P_{\rm form}$ の block-orthogonality による exact cancellation と $P_{\rm int}^{\rm MX}$ の χ-specialisation については、本稿では symbolic computation による確認結果として提示し、formal lemma / proposition の形式での抽象的定式化までは踏み込まない。

### 8.4 Closing remarks

以上を要約すると、本稿は Lorentzian EC+NY reduced sector において、 $S^3/T^3/Nil^3/Sol^3$ の四 topology が共通の admissibility pattern を持ち、 topology dependence が diagnostic ( $C_{\rm topology}=-9\chi$ ) と reduced vacuum orbit ( $\dot q^2+\chi=0$ ) の二つの独立な経路で同一 scalar $\chi$ に集約されることを示した。この $\chi$ -controlled topology universality は reduced sector における分類結果であり、 anisotropic / matter-coupled / global 拡張への一般化は本稿の範囲を超える今後の課題である。
